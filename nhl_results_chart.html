How Do I Read This Chart?

Each square corresponds to a single 2019-2020 regular-season NHL game. Each Atlantic division team has a row, and each opponent has a column.

  • A green square is a win for the row team (2 points)
  • A yellow square is an overtime loss for the row team (1 points)
  • A red square is a regulation loss for the row team (0 points)
  • An empty square is a game yet to be played
